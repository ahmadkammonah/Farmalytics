WHAT WE DO?

This application provides a comprehensive tool to visualize geospatial data using an interactive map. You can draw polygons, view different map layers, and interact with the groundwater runoff data provided in GeoJSON format. Our goal is to make data visualization and geospatial analysis accessible and intuitive for all users.
